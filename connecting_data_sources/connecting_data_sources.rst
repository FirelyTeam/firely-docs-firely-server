Connecting Data Sources
=======================

Firely Server provides multiple ingestion patterns for integrating external data sources. 
This section covers how to configure bulk and event-driven ingestion using Firely Server Import (FSI), PubSub, and Facade, and how these mechanisms differ in terms of throughput, processing model, and integration scenarios.
