Features - Clinical & Quality Workflows
=======================================

Firely Server supports advanced clinical workflows and quality measurement use cases. 
Digital Quality Measures enable standardized evaluation of care quality using computable logic, while CDS Hooks allows real-time clinical decision support to be integrated directly into provider workflows. 
Together, these capabilities help drive data-informed care and improve clinical outcomes.
