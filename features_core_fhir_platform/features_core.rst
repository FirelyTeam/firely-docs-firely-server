Features - Core FHIR platform
=============================

At the heart of Firely Server is a fully compliant FHIR RESTful API that enables standardized access to healthcare data. 
Incoming resources are validated against the core FHIR specification and Implementation Guides to ensure data quality and conformance. 
Firely Server is highly extensible, supporting custom resources, search parameters, and operations to meet specific use cases. 
Additionally, an OpenAPI definition provides clear, machine-readable API documentation, making integration and development more efficient.
