Features - Data Exchange
========================

Firely Server provides robust capabilities for exchanging healthcare data across systems and workflows. 
It supports standardized document generation and handling via FHIR, including the $document operation, as well as efficient management of binary content. 
Bulk Data Export enables performant extraction of datasets for population-level use cases. 
Additionally, Subscriptions allow systems to react to data changes in real time, supporting event-driven integration patterns.
