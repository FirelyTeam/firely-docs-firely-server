Programming API Reference & Plugins
===================================

Firely Server offers an extensive programming api which allows changing the behaviour of the server.
Custoomizations can be implemented using a plugin infrastructure.
The reference documentation lists the available plugins for configuring the pipeline, and the public programming API of Firely Server for building Plugins and Facades.
